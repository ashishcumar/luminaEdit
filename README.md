# LuminaEdit - Browser-Based Video Editor

Professional web-based video editor built with React, TypeScript, and FFmpeg.wasm.

## Features
- Interactive Timeline with Drag & Drop
- Video Trimming and Volume Control
- Visual Filters (Grayscale, Sepia, etc.)
- Text Overlays with Custom Styling
- Efficient Video Compression
- Client-side Exporting using FFmpeg

## Development
```bash
npm install
npm run dev
```

## Production Build
```bash
npm run build
```
